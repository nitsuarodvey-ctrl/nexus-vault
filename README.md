# Nexus Vault | Developer Component Library hub 🏛️

[TR] Nexus Vault, web projelerinizde sıkça kullandığınız HTML/JSX bileşenlerini (kod parçacıklarını) tek bir merkezde toplamanızı, anlık olarak önizlemenizi ve projelerinize hızlıca entegre etmenizi sağlayan bir "Developer Vault" projesidir.
[EN] Nexus Vault is a "Developer Vault" project that allows you to centralize your frequently used HTML/JSX components (snippets), preview them instantly, and quickly integrate them into your web projects.

---

## ✨ Özellikler / Features

### 🇹🇷 Türkçe
- **Canlı Önizleme (Live Preview):** Eklediğiniz bileşenleri gerçek zamanlı olarak canvas üzerinde görün.
- **Dinamik Kütüphane Yönetimi:** LocalStorage entegrasyonu ile kendi bileşenlerinizi ekleyin, kategorize edin ve saklayın.
- **Kod Yönetimi:** Fira Code fontu ile optimize edilmiş kod alanı ve tek tıkla kopyalama özelliği.
- **Modern UI:** Tailwind CSS ve Glassmorphism prensipleriyle tasarlanmış karanlık mod arayüzü.

### 🇺🇸 English
- **Live Preview:** View your components in real-time on a dedicated preview canvas.
- **Dynamic Library Management:** Add, categorize, and store your own components using LocalStorage integration.
- **Code Management:** Optimized code display with Fira Code font and one-click copy functionality.
- **Modern UI:** Dark mode interface designed with Tailwind CSS and Glassmorphism principles.

---

## 🛠️ Teknik Yığın / Tech Stack

- **Architecture:** Vanilla JavaScript (State Management via LocalStorage)
- **Styling:** Tailwind CSS, Glassmorphism UI
- **Typography:** Plus Jakarta Sans & Fira Code (Google Fonts)
- **Icons:** Material Symbols Outlined

---

## 🚀 Kullanım / How to Use

[TR] Herhangi bir kuruluma ihtiyaç duymaz. `index.html` dosyasını tarayıcıda çalıştırarak kendi bileşen kütüphanenizi oluşturmaya başlayabilirsiniz.
[EN] No setup required. Just open `index.html` in your browser to start building your personal component library.
